# Conversational AI BI Dashboard

Final Year Project – Master of Data Science (RMIT)

## 🚀 Overview
This project builds a conversational AI-powered business intelligence system that allows SMEs to query their data using natural language.

## 🧠 System Flow
User Query → NLP → SQL → Database → Analytics → Explanation → Dashboard

## 🛠️ Tech Stack
- FastAPI (Backend)
- PostgreSQL (Database)
- OpenAI API (NLP)
- Pandas / scikit-learn (Analytics)
- Streamlit (Frontend)
